Detailed Description

Display control command.

Sets entire display (D) on/off, cursor on/off (C), and

blinking of cursor position character (B).

#define C_POSITION   1

Cursor enable bit position. The cursor is displayed when C is 1 and.

of I/D or other specifications will not change during display data write.
